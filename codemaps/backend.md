# Backend Codemap

**Last Updated:** 2025-02-23
**Runtime:** Next.js 15.2.6 (Static Export)
**Build Mode:** Static Site Generation (SSG)
**Output:** Static HTML/CSS/JS

## Overview

This is a **static site** with minimal backend. All content is pre-rendered at build time. The only dynamic functionality is handled through:

1. **API Routes:** Newsletter subscription
2. **Third-Party Services:** Comments (Giscus), Analytics (Google)
3. **Build Scripts:** RSS generation, sitemap

## API Routes

### Newsletter API

```
app/api/newsletter/route.ts
```

**Methods:** GET, POST

**Provider:** Buttondown (via Pliny)

**Implementation:**
```typescript
import { NewsletterAPI } from 'pliny/newsletter'
import siteMetadata from '@/data/siteMetadata'

const handler = NewsletterAPI({
  provider: siteMetadata.newsletter.provider  // 'buttondown'
})

export { handler as GET, handler as POST }
```

**Configuration:**
- Environment variables handled by Pliny
- Buttondown API key required
- Static export: `dynamic = 'force-static'`

**Request/Response:**
- Handles email subscription form
- Returns JSON response
- Error handling included

## Build Process

### Build Pipeline

```
yarn build
    ↓
next build (Static Export)
    ↓
Contentlayer Import
    ├─ Parse MDX files
    ├─ Generate computed fields
    ├─ Create JSON index
    └─ Write .contentlayer/generated/
    ↓
Next.js Page Generation
    ├─ Static routes
    ├─ Dynamic routes (pre-rendered)
    └─ API routes
    ↓
node scripts/postbuild.mjs
    ↓
RSS Generation
    ├─ Main feed (feed.xml)
    ├─ Tag feeds (tags/*/feed.xml)
    └─ Write to build/ or public/
    ↓
Output: build/ directory
```

### RSS Generation Script

**File:** scripts/rss.mjs

**Functions:**
- `generateRssItem(config, post)` - Generate RSS item XML
- `generateRss(config, posts, page)` - Generate RSS feed XML
- `generateRSS(config, allBlogs, page)` - Write RSS files

**Output:**
1. **Main Feed:** `build/feed.xml`
   - All published posts, sorted by date
   - RSS 2.0 format

2. **Tag Feeds:** `build/tags/{tag}/feed.xml`
   - Posts with specific tag
   - One feed per tag

**RSS Item Structure:**
```xml
<item>
  <guid>{siteUrl}/blog/{slug}</guid>
  <title>{post.title}</title>
  <link>{siteUrl}/blog/{slug}</link>
  <description>{post.summary}</description>
  <pubDate>{post.date}</pubDate>
  <author>{email} ({author})</author>
  <category>{tag}</category>
</item>
```

## Content Generation

### Sitemap Generation

**File:** app/sitemap.ts

```typescript
export default function sitemap(): MetadataRoute.Sitemap {
  const siteUrl = siteMetadata.siteUrl

  // Static routes
  const routes = ['', 'blog', 'favorites', 'tags'].map(route => ({
    url: `${siteUrl}/${route}`,
    lastModified: new Date().toISOString().split('T')[0]
  }))

  // Blog routes (published only)
  const blogRoutes = allBlogs
    .filter(post => !post.draft)
    .map(post => ({
      url: `${siteUrl}/${post.path}`,
      lastModified: post.lastmod || post.date
    }))

  return [...routes, ...blogRoutes]
}
```

**Output:** `/sitemap.xml` (auto-generated by Next.js)

### Search Index Generation

**Trigger:** Contentlayer onSuccess callback

**File:** Generated in contentlayer.config.ts

```typescript
function createSearchIndex(allBlogs) {
  if (siteMetadata?.search?.provider === 'kbar') {
    writeFileSync(
      `public/${path.basename(siteMetadata.search.kbarConfig.searchDocumentsPath)}`,
      JSON.stringify(allCoreContent(sortPosts(allBlogs)))
    )
  }
}
```

**Output:** `public/search.json`

**Indexed Fields:**
- title
- slug
- date
- summary
- tags
- path

### Tag Data Generation

**Trigger:** Contentlayer onSuccess callback

**File:** contentlayer.config.ts

```typescript
function createTagCount(allBlogs) {
  const tagCount: Record<string, number> = {}

  allBlogs.forEach(file => {
    if (file.tags && (!isProduction || file.draft !== true)) {
      file.tags.forEach(tag => {
        const formattedTag = slug(tag)
        tagCount[formattedTag] = (tagCount[formattedTag] || 0) + 1
      })
    }
  })

  writeFileSync('./app/tag-data.json', JSON.stringify(tagCount))
}
```

**Output:** `app/tag-data.json`

**Format:** `{"tag-slug": count}`

### Regular Tasks

1. **Content Updates:** Add MDX files to `data/blog/`
2. **Build:** Run `yarn build` to regenerate

4. **Verify:** Check sitemap, RSS, search index

### Build Troubleshooting

**Issue:** Contentlayer import fails
- Check MDX frontmatter syntax
- Verify contentlayer.config.ts

**Issue:** RSS not generated
- Check postbuild.mjs execution
- Verify tag-data.json exists

**Issue:** Search not working
- Verify search.json in public/
- Check kbar configuration
