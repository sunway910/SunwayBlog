## 工作

- ### 1 Http Request Header

**在开发的时候发现一个有意思的地方**

> During this process, the HTTP header key is normalized based on the net/http standard library in Go. While the header keys themselves are case-insensitive, the normalization process capitalizes the first letter of the header and each of the hyphenated words within the header.

- 前端在使用 `axios` 构建自定义 `http header` 请求时，会将 `custom-header` 转换为 `Custom-Header`
- `golang gin server` 在使用 `c.Request.Header.Get("custom-header")` 获取请求头时，也会自动转换为 `Custom-Header`

- ### 2 前端工作流更新

这周承接了一个前端开发的工作，以下是我的十倍界王拳开发工作流
- 1. 拿到设计给的设计图，拉扯讨论定稿
- 2. 将设计稿的截图喂给 `V0`, 给出 `prompt`，比如使用 `nextjs/typescript/react/tailwindcss/shadcn-ui components...` 实现
- 3. 从 V0 拿到初版的代码后，开始喂给 `claude sonnet 3.7` 开始按照个性化需求开始调整，然后开始摸鱼
